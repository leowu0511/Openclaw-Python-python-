# 🚀 OneClick-Openclaw-Win

**還在為了配 Node.js 環境、找 OpenRouter 免費模型、或是搞不定 Tier 1 權限而煩惱嗎？** 這是一個專為 Windows 用戶設計的「懶人一鍵部署工具」。
只要下載一個 EXE，我們幫你搞定所有瑣事。

---

### ✨ 核心功能
* **📦 零環境依賴**：內建自動下載 Node.js v22 與 MinGit，不汙染你的主機系統。
* **🛠️ 一鍵修復**：自動 Patch `vite.config.ts` 路徑，解決 UI 無法啟動的頭號難題。
* **🤖 2026 戰神模型清單**：預設收錄 15+ 個實測有效的真實免費模型（如 Trinity Large, Hunter Alpha）。
* **💬 Discord 綁定**：在設定時輸入 Token，直接把 OpenClaw 變身 Discord 機器人。
* **💰 Tier 1 管理**：協助你達成 $10 儲值門檻，解鎖每日 1,000 次 API 額度。

---

### 📥 快速啟動 (只需 3 步)
1. **下載**：到 [Releases] 下載最新的 `manager_openrouter_win.exe`。
2. **填寫**：執行程式，根據提示貼上你的 `OpenRouter API Key`。
3. **起飛**：選擇你要的模型，按下 Enter，OpenClaw 就跑起來了。

> **💡 專業提示**：如果你想讓筆電省電，建議遠端連回你的 3060 Ti 電腦跑這套工具。

---

### 🛡️ 安全性說明
* **隱私至上**：API Key 與 Discord Token 僅儲存在你本地的 `.openclaw/` 資料夾，絕不外傳。
* **環境隔離**：所有下載的依賴項目均為綠色免安裝版，隨時可以整包刪除。
